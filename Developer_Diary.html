July 19, 2017 - SFTP All The Files!!! & Mailto Link

I've spent the better part of a week working to set up an SFTP to the UMich file hosting website... I originally wrote a python script to push all the files, but I was having an issue due to the password that was required to connect to a server. I ended up just using a SFTP plugin on Sublime to set up the SFTP to the server. It was easier than I expected.

I have also been working on improving some basic things on the website. The first thing I fixed was the mail link. Originally it was just my email that I had when one looked at the contact page. Now, when someone clicks on the email it will automatically set up a mailto link. This was pretty easy to do, I just set the href to "mailto:example@email.com"

August 6, 2017 - A Different Approach To Learning

Two websites whose email list I subscribe to (and look forward to) are medium and tech crunch. They are where I receive the majority of my news and daily readings from. Recently, I read a post on medium.com that I really liked, and I wanted to talk about how it has affected how I am going to approach the design of this website.

The post I am talking about talks about how new web developers approach learning. I personally have fallen into the issue of trying to perfect every part of the website. As a result, I haven't been progressing as much as I'd like to. From here on out I am not going to try and be as much of a perfectionist on my website, rather I'd like to add everything to my website before I start perfecting things.
